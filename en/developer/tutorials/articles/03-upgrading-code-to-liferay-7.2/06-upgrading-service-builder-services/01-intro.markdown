# Upgrading Service Builder Services

[TOC levels=1-4]

To properly upgrade app's leveraging service builder, you must complete the
following steps:

- Remove Legacy Files
- Migrate from Spring DI to OSGi Declarative Services
- Rebuild Services

You'll start by removing legacy files.
